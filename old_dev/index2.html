Binary Battler

A red-vs-blue space shooter built with JavasScript, HTML and CSS

Binary Battler was my first independent game project. I designed the game using JavaScript, HTML and CSS to build a side-by-side shooting game that tests players' reflexes. Even without the Canvas framework, the game features collision detection, key press responses, animations, progress bar interaction and depletion, color toggles as classes, and a retro 80s-inspired layout. So I guess you could say I play life in hard mode

Language Cafe

Language exchange forum built using Ruby on Rails and Bootstrap

Language Cafe is a Ruby on Rails forum for people who want to start a language exchange. Annoyed by the general bulkiness of navigating Meetup.com, I designed this Rails app to specifically target language learners and instructors. The back end features route and model nesting, making it easy for users to see the path they took based just on the URL. On the front end, the app was designed to be mobile-first and thus features responsive design as well as a simple responsive fixed-top navbar.

You can check out Language Cafe here.

SnapThis

A photo-sharing scavenger hunt developed using the MEN stack and Sockets for real-time fun

SnapThis was designed to be mobile-first so players could run around the city to snap and send photos to the host. Host and player alike are required to sign in via Google's OAuth service. Hosts have full CRUD functionality over tasks, and players can submit images via imageUrl functionality. The design was built using Google's Materialize CSS framework.

You can sign up to play SnapThis here! Click click!.

Resettlement Hub

SPA built with MEAN stack

Resettlement Hub is built around the demand that refugee resettlement organizations have for volunteers. All users sign up using JavaScript's jwt authentication service. The app was built using the MEAN stack following Single-Page App principles. Moving forward, I plan to upgrade this to Google's OAuth service to streamline the signup process for volunteers and nonprofit organizations. Other upcoming features include full profile for organizations and volunteers, and a migrant crisis-relevant newsfeed pulling from an external API. Come check out the site here and start listing your agency's events for volunteers to see!
